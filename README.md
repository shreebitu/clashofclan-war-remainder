# Clash of Clans War & Member Tracker

A simple static web page to track **Clash of Clans** wars and clan members, created by **UN1VERSE**. This project is a starting point for building a comprehensive clan management tool, with plans to integrate backend features in the future.  

---

## Features (Current)

- Track wars for your clan in a simple and organized way.  
- Monitor clan members’ participation and status.  
- Lightweight and static design for fast access.  

---

## Planned Future Features

**UN1VERSE** aims to expand this tracker with powerful features, including:  

- **Clan Tag Input:** Fetch complete member details for any clan by simply entering the clan tag.  
- **Player ID Lookup:** Retrieve troops, equipment, and levels of any player by entering their player ID.  
- **Detailed Stats:** Get comprehensive information about each member’s war readiness, troop levels, and more.  

> Note: These features are under development and will take time to implement.  

---

## How to Use

1. Open the web page: [Clash of Clans War & Member Tracker](https://shreebitu.github.io/clashofclan-war-remainder/)  
2. Input your clan tag or player ID (future feature).  
3. View war details and member statistics.
